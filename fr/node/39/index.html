Call to undefined function image_gd_check_settings()

Getting Fatal error: Call to undefined function image_gd_check_settings() in ...

This was after I migrated a Drupal site to Aegir.

The solution is to visit this page:

admin/settings/image-toolkit

and click save.

See also http://drupal.org/node/295854
