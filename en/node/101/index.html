Comment se faire des clés API Facebook et Twitter

Submitted by Albert on Wed, 03/05/2014 - 16:52

Voici comment se faire des clés API Facebook et Twitter. Vous en aurez besoin pour utiliser les services de connexion à votre site via ces comptes.

Pour cet exemple, disons que vous développez un site "Outils AAA" et que votre environnement de développement est http://outils.local/

  • Connectez-vous sur votre compte Twitter
  • Aller au https://apps.twitter.com/app/new
  • Name: mettez un nom unique, par exemple "Outils AAA"
  • Description: description de votre site
  • Website: http://outils.local/
  • Callback URL: http://outils.local/
  • Créez votre application
  • Allez dans l'onglet Settings et cochez "Allow this application to be used to Sign in with Twitter"

  • Allez dans l'onglet API Keys, vous y trouverez les clés d'API.

  • Connectez-vous à Facebook.

  • Allez à https://developers.facebook.com
  • Si c'est la première fois que vous enregistrez un site avec ce systeme: dans le menu Apps, sélectionnez "Register as a developer" (il vous faut accès à un téléphone pour obtenir un code).
  • Une fois que c'est fait, dans le menu Apps, ajouter une Application.
  • Display name: mettez par exemple "Outils AAA"
  • Namespace: mettez "outils_aaa"
  • Dans les "settings" de votre app, cliquez sur "Add platform" et ajoutez un "Website".
  • Dans la section Website, toujours sur Facebook, mettez votre URL, par example: http://outils.local/. Dans App Domains mettre la même chose. Si vous ne faites pas ceci vous aurez une page blanche quand vous essaierez de vous connecter via Facebook. Si vous avez plus d'un nom de domaine, voir cet article. Votre site URL est le URL principal, et les App Domains sont tous les domaines que vous utilisez, par exemple: fr.monsite.dev, fr.monsite.local, en.monsite.dev, en.monsite.local, etc.

À présent vous pourrez utiliser ces clés API pour permettre des connexions à votre site web via Facebook et Twitter. Voici comment le faire sur Drupal.
